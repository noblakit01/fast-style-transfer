# tf-coreml converter installation

## Dependencies:
  * tensorflow >= 1.5.0
  * coremltools >= 0.8
  * numpy >= 1.6.2
  * protobuf >= 3.1.0
  * six >= 1.10.0

## Installation
* Install From PyPi:
```
pip install -U tfcoreml
```
